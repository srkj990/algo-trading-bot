Repository Capabilities Overview
This is a Zerodha Algo Trading Bot designed for automated equity trading with flexible configurations and multiple data/execution providers. Here's what it can do:

🎯 Core Trading Capabilities
Two Distinct Trading Engines:

Intraday Equity: Trades during market hours (9:15 AM - 3:15 PM) with 1-minute candles using MIS (Margin Intraday Square-off) product
Delivery Equity: Overnight/positional trading with 1-day candles using CNC (Cash and Carry) product
📊 Trading Strategies
The bot supports multiple technical analysis strategies:

Moving Average (MA): 20/50 cross-over strategy
RSI: Relative Strength Index (oversold < 30, overbought > 70)
Breakout: 20-candle high/low breakout detection
VWAP: Volume-Weighted Average Price strategy
ORB (Opening Range Breakout): Intraday only - breakout from first 15-minute range
🔄 Signal Generation Modes
Single Strategy Mode: Evaluates one strategy at a time
Multi-Strategy Mode: Combines multiple strategies with configurable minimum confirmations for consensus-based signals
Ranked Candidate Selection: Scores and ranks signals, enters top N positions
💰 Risk Management
ATR-Based Position Sizing: Calculates quantity based on Average True Range and capital-at-risk
ATR-Based Stop-Loss: Dynamic stop-loss using ATR multiples (no fixed percentages)
Three Risk Profiles:
Conservative (ATR: 1.5x, Risk: 0.5%)
Balanced (ATR: 2.0x, Risk: 1.0%)
Aggressive (ATR: 2.5x, Risk: 1.5%)
Trailing Stops: Follows price movement to lock in profits
Target Prices: Risk-reward ratio based exit targets (1.8:1 to 2.2:1)
Position Limits: Control max open positions and capital allocation per trade
📈 Data Providers
Fetch market data from:

YFinance: Public Yahoo Finance data (no auth required)
Zerodha Kite: Connected through Kite Connect API
Upstox: Connected through Upstox APIs
🚀 Execution Modes
Paper Trading: Simulates trades without executing
Live Trading: Actual order placement with position sync
🔗 Execution Providers
Zerodha Kite: Order placement, position tracking, holdings retrieval
Upstox: API-based order placement and portfolio management
📊 Backtesting Engine
Standalone daily-bar backtesting on NIFTY50 universe
Configurable capital, risk parameters, and position sizing
Generates equity curves and trade summaries
Supports multiple strategies with multi-strategy consensus
🔐 State Management
Persists position state to state directory
Tracks open positions, entry prices, stop-losses, and targets
Survives restarts by reloading saved state
⚙️ Flexible Configuration
Interactive setup prompts for:

Data provider selection
Paper/Live mode
Execution provider choice
Engine type (Intraday/Delivery)
Capital allocation
Symbol universe (Manual list or NIFTY50)
Risk style & parameters
Position limits
Entry selection strategy
Strategy mode (single/multi)
📱 Supported Features
Multiple Symbol Sets: NIFTY50 scan or manual symbol lists
Market Hours Management: Respects trading hours, market holidays/weekends
Gap Detection: Intraday engine detects and handles opening gaps
Volume Filters: Breakout strategy uses volume multipliers (1.2x)
Logging & Audit: Comprehensive event logging for debugging
